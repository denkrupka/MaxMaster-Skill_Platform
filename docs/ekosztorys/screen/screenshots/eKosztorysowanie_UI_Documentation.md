## 2. Шапка приложения (Header)

### 2.1 Логотип и название
```
eKOSZTORYSOWANIE
```
- Расположение: Левый верхний угол
- Цвет: Белый текст на синем фоне
- "e" и "OWANIE" - обычный шрифт
- "KOSZTORYS" - выделено жёлтым/золотым

### 2.2 Навигация слева

#### 2.2.1 Иконка "Домой" (🏠)
- При нажатии: Переход на главную страницу/список кошторисов

#### 2.2.2 Dropdown "Kosztowys"
**Внешний вид**: Кнопка с текстом текущего кошториса и стрелкой ▼

**При нажатии**: Открывается список доступных кошторисов для переключения

#### 2.2.3 Кнопка "+" (Создать новый)
- При нажатии: Создание нового кошториса

### 2.3 Иконки справа

| Иконка | Название | Функция |
|--------|----------|---------|
| ? | Помощь | Открывает справку |
| 📷 | Скриншот | Создание снимка экрана |
| 🚩 | Флаг | Отметить/закладка |
| 📋 | Буфер | История буфера обмена |
| 👤 | Профиль | Настройки пользователя |
| 📱 | Rozszerzenia | Расширения/плагины |

## 3. Главная панель инструментов (Toolbar)

### 3.1 Первая строка (основные действия)

#### 3.1.1 Кнопка "☰ Wydruki" (Печать/Экспорт)
**Расположение**: Левый край первой строки

**При нажатии**:
- Открывается панель "Eksportuj kosztorys" слева
- Показываются шаблоны печати
- Появляется предпросмотр документа

**Связанные скриншоты**: 0001-0048

#### 3.1.2 Иконка монитора (📺)
**Функция**: Переключение режима отображения/предпросмотр

#### 3.1.3 Кнопка "Drukuj"
**При нажатии**:
- Если панель Wydruki открыта → Открывает системный диалог печати
- Если панель закрыта → Сначала открывает панель Wydruki

#### 3.1.4 Кнопка "📝 Komentarze"
**При нажатии**: Открывает панель комментариев справа

**Связанные скриншоты**: 0152-0171

### 3.2 Вторая строка (редактирование)

Полный набор кнопок слева направо:

```

#### 3.2.1 Кнопка "☰ Kosztorys"
**При нажатии**: Переключает вид на режим "Kosztorys" (основная таблица сметы)

**Связанные скриншоты**: 0058

#### 3.2.2 Кнопка "+ Dział" (с dropdown ▼)
**При нажатии на кнопку**:
- Добавляет новый раздел (Dział) в смету
- Раздел добавляется после текущего выбранного элемента

**При нажатии на ▼** - Выпадающее меню:
```
┌─────────────────────────────┐
│ + Dział                     │  ← Добавить раздел
│ + Poddział                  │  ← Добавить подраздел
└─────────────────────────────┘
```

**Связанные скриншоты**: 0089

#### 3.2.3 Кнопка "KNR Pozycja" (с dropdown ▼)
**При нажатии на кнопку**:
- Открывает браузер каталогов KNR/KNNR
- Позволяет выбрать позицию из каталога

**При нажатии на ▼** - Выпадающее меню:
```
┌─────────────────────────────────────┐
│ KNR Pozycja                         │  ← Позиция из каталога
│ Wstaw pozycję nieskatalogowaną      │  ← Некаталогизированная позиция

```

**Связанные скриншоты**: 0101-0109

**Структура браузера KNR**:
- Иерархическое дерево каталогов
- Поиск по коду (например: "KNNR 5 0701-01")
- Предпросмотр описания позиции
- Кнопка "Wstaw" для вставки

#### 3.2.4 Кнопка "📋 Nakład" (с dropdown ▼)
**При нажатии**: Добавляет накладную к выбранной позиции

**Типы накладных**:
- **R** (Robocizna) - Трудозатраты
- **M** (Materiały) - Материалы
- **S** (Sprzęt) - Механизмы/Оборудование
- **O** (Odpady) - Отходы

#### 3.2.5 Кнопка "Uzupełnij nakłady" (с dropdown ▼)
**При нажатии**:
- Автоматически загружает недостающие накладные из каталога KNR
- Применяется к выбранной позиции

**Связанные скриншоты**: 0120

#### 3.2.6 Кнопка "Ceny"
**При нажатии**:
- Открывает модальное окно "Uaktualnij ceny w kosztorysie"

#### 3.2.7 Кнопка "📝 Komentarze" (с dropdown ▼)
**При нажатии на кнопку**: Открывает панель комментариев

**При нажатии на ▼** - Выпадающее меню:
```

**Связанные скриншоты**: 0174-0177

#### 3.2.9 Кнопка "↕ Przesuń" (с dropdown ▼)
**При нажатии на ▼** - Выпадающее меню:
```
┌───────────────────────────────────────────┐
│ ↑ Przesuń pozycję w górę                  │  ← Переместить вверх
│ ↓ Przesuń pozycję w dół                   │  ← Переместить вниз
│ ↗ Przesuń pozycję do pierwszego działu    │  ← Переместить в первый раздел
└───────────────────────────────────────────┘
```

**Связанные скриншоты**: 0178-0187

**Эффект каждого действия**:
- **Przesuń w górę**: Позиция перемещается выше (номер Lp уменьшается)
- **Przesuń w dół**: Позиция перемещается ниже (номер Lp увеличивается)
- **Przesuń do pierwszego działu**: Позиция выносится в первый раздел сметы

#### 3.2.10 Кнопка "📋 Kopiuj"
**При нажатии**:
- Копирует выбранный элемент в буфер обмена
- Появляется toast-уведомление об успешном копировании
- Элемент остаётся на месте

**Связанные скриншоты**: 0189-0192

#### 3.2.11 Кнопка "✂ Wytnij"
**При нажатии**:
- Вырезает выбранный элемент
- Элемент визуально затеняется
- Активируется кнопка "Wklej (wycięta pozycja)"

**Связанные скриншоты**: 0198

#### 3.2.12 Кнопка "📋 Wklej"
**При нажатии**:
- Вставляет элемент из буфера
- Если был Wytnij → перемещение
- Если был Kopiuj → копирование

**Состояния кнопки**:
- `Wklej` - обычное состояние
- `Wklej (wycięta pozycja)` - когда есть вырезанный элемент
- `Wklej (pozycja)` - когда есть скопированная позиция

**Связанные скриншоты**: 0195, 0199

## 14. Формулы расчётов

### 14.1 Расчёт стоимости накладной

```
Wartość nakładu = Ilość × Norma × Cena jednostkowa
```

Пример из скриншота 0099:
```
robotnicy (R): 10,0 × 1,0769000 × 52,86 = 569,18 PLN
Koparka (S): 10,0 × 16,9000000 × 11,00 = 1859,00 PLN
```

### 14.2 Расчёт стоимости позиции

```
Wartość pozycji = Σ(Wartość nakładów) + Narzuty
```

### 14.3 Формула Przedmiar

```
Przedmiar = a × b × c × d × ... × Krotność
```

Пример:
```
1.1 × 1.1 × 1 × 0.89 = 1,0769 m³
```

### 14.4 Накрутки (Narzuty)

```
Kp (Koszty pośrednie) = (R + S) × Stawka_Kp%
Z (Zysk) = (R + S + Kp) × Stawka_Z%
Kz (Koszty zakupu) = M × Stawka_Kz%
```

### 14.5 Итоговая стоимость

```
Wartość kosztorysu = Σ(Wartość pozycji) + Σ(Narzuty)
```

Из скриншотов: **2,428.18 PLN** → **3,286.19 PLN** (после изменений)

### 15.3 Схема базы данных (PostgreSQL)

#### Основные таблицы

```sql
-- Сметы
CREATE TABLE cost_estimates (
    id UUID PRIMARY KEY,
    type VARCHAR(20) NOT NULL,  -- 'investor', 'contractor', 'offer'
    name VARCHAR(300) NOT NULL,
    currency VARCHAR(3) DEFAULT 'PLN',

    -- Глобальные коэффициенты
    factor_labor DECIMAL(10,6) DEFAULT 1,
    factor_material DECIMAL(10,6) DEFAULT 1,
    factor_equipment DECIMAL(10,6) DEFAULT 1,
    factor_waste DECIMAL(10,6) DEFAULT 1,

    -- Итоги
    total_value DECIMAL(15,2) DEFAULT 0
);

-- Разделы
CREATE TABLE sections (
    id UUID PRIMARY KEY,
    cost_estimate_id UUID REFERENCES cost_estimates(id),
    parent_section_id UUID REFERENCES sections(id),
    ordinal_number INTEGER NOT NULL,
    name VARCHAR(300)
);

-- Позиции
CREATE TABLE positions (
    id UUID PRIMARY KEY,
    cost_estimate_id UUID REFERENCES cost_estimates(id),
    section_id UUID REFERENCES sections(id),
    base VARCHAR(50),           -- "KNNR 5 0701-01"
    name TEXT NOT NULL,
    unit_label VARCHAR(20) NOT NULL,
    measurements JSONB,
    quantity DECIMAL(15,6) DEFAULT 0
);

-- Ресурсы
CREATE TABLE resources (
    id UUID PRIMARY KEY,
    position_id UUID REFERENCES positions(id),
    type VARCHAR(20) NOT NULL,  -- 'labor', 'material', 'equipment'
    name VARCHAR(300) NOT NULL,
    norm_value DECIMAL(15,6) NOT NULL,
    unit_price DECIMAL(15,2) DEFAULT 0
);
```
